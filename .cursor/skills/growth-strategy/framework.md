# Growth Strategy Framework (Condensed)

## Foundation

- Growth = Retention + Acquisition + Monetization (cohesive system, not silos)
- System must be: Systematic, Deterministic, Sustainable, Repeatable
- Retention is the power plant; it drives acquisition and monetization

## Phase 1: Retention

- Use case: Problem, Persona, Alternative + Natural Frequency, Core action
- Retention metric: Binary (active or not); inputs = Activation, Engagement, Resurrection
- Avoid: Wrong frequency, wrong core action, wrong audience
- Cohorts and retention curves for visualization

## Phase 2: Engagement & Habit Loops

- Organic habit loop: Trigger (from problem) → Action (core action) → Reward (why choose you)
- Manufactured loops: Push, email, in-app
- Environment loops: Physical/environmental cues
- Activation: Setup moment, Aha moment, Habit moment
- Resurrection: Involuntary (reminders) vs voluntary (value, incentives)

## Phase 3: Acquisition

- Loops, not funnels. Loop = Input → Steps → Output (reinvested)
- Categories: Viral, Content (UGC), Paid, Sales
- Viral loop blueprint: user gets value → product encourages sharing → shared artifact attracts new users → new users enter loop (tailored to product). Referral mechanics: incentive structure, sharing friction, tie to core action/habit loop.
- Channel tactics (where relevant): Organic (SEO/content, social, community); Paid (LTV:CAC, creative testing, retargeting); Product (in-product referral, UGC, API); Partnership (integrations, co-marketing, affiliates).
- Principles: Channel dynamics (habit transfer), Product-channel fit, Model-channel fit
- Levers: Optimize loops, Add loops (order matters), Linear (sparingly)

## Phase 4: Monetization

- Four components: How, When, What, Amount (each adds friction)
- Four Fits: Product-market, Product-channel, Model-channel, Model-product, Model-market
- Pricing: Value metric, structure, amount
- Balance: Perceived Value vs Perceived Price + Friction

## Phase 5: Growth Model

- Qualitative: Output, habit loops, acquisition loops, linear
- Loop-specific quant: Single loop impact
- End-to-end quant: All components together
- Use for: Predictions, goal-setting, prioritization, communication, updates

## Phase 6: User Psychology (ELMR + Psych)

- ELMR: Emotion (fuel), Logic (justification), Motivation (ability + drive), Reward (confirmation)
- Psych: Fuel tank; negative psych (consumes), positive psych (adds)
- For each flow step: assign +/- psych; find drains and gaps
- Boosts: Urgency, scarcity, bargain, belonging

## Phase 7: Experiments

- Hypothesis: Problem → Belief → Evidence → Predictions
- MVT: One question, valid design, success metric, tradeoff metric
- Prioritize: ICE (Impact 1-5, Confidence 1-5, Ease 1-5); priority tier (Quick win, High impact, Moonshot); experiment prioritization table
- Apply learnings to growth model and user psychology map

## Phase 8: Defensibility

- Network effects, Brand, Embedding, Scale
- Strengthens loops; makes tactics harder to copy

## Execution and Metrics

- **Key metrics to track**: By phase (acquisition, activation, retention, referral, revenue); recommend 1-2 dashboard views (e.g. acquisition funnel, retention cohorts, experiment pipeline).
- **ICE**: Use for experiment prioritization (Impact, Confidence, Ease) and include prioritization table in output.
- **Viral loop blueprint**: For acquisition, spell out: user gets value → product encourages sharing → shared artifact attracts new users → new users enter loop (tailored to product).
- **6-week cadence** (optional): Weeks 1-2 analysis and roadmap; Weeks 3-4 experiments; Weeks 5-6 scale and monitoring.
